這是一款使用 **純原生 HTML5 Canvas + Vanilla JavaScript** 實作的經典縱向捲軸射擊街機遊戲。專案專為高相容性與離線執行設計，完全沒有任何外部圖片、音效資源或第三方框架依賴，解除了在 Windows 本地開啟檔案時的瀏覽器跨域（CORS）安全性限制，實現 **100% 雙擊即玩**。

---

## 🎮 遊戲特點 (Features)

### 1. 銳利向量與流暢動畫 (Vector & Loop Engine)
* **高級向量戰機外觀**：採用多層次路徑與漸層畫筆，手動繪製出精美的雙尖端前掠翼戰機與黃橙漸層座艙罩。
* **旋轉六角能量護盾**：以矩陣旋轉演算法，繪製呼吸閃爍、半透明的旋轉六角形防護罩。
* **原生極輕量粒子引擎**：
  * **雙引擎尾焰**：戰機尾部雙噴氣口與機翼安定翼配有流動、淡化縮放的青色與桃紅色粒子尾跡。
  * **霓虹碎屑爆炸**：敵機被摧毀時向四周濺射出對應色彩（綠/桃紅/橘/紅）的爆發粒子。
  * **量子炸彈全屏粒子**：釋放炸彈時，全螢幕隨機漂移發光的粉紫粒子，伴隨強烈的相機震屏（`ctx.translate` 偏移）。

### 2. 即時 Web Audio API 音效合成器 (Procedural Audio Synth)
遊戲中所有的背景音樂 (BGM) 與音效皆由瀏覽器即時合成運算，免去載入音樂檔的時間與網路開銷：
* **音效 (SFX)**：發射雷射聲（頻率指數衰減）、護盾與裝甲受擊聲、白噪音爆炸聲、升級音效、核彈引爆低音。
* **背景音樂 (BGM)**：實作了一首 160BPM 節奏感的 **Synthwave 復古電子樂**（包含低音貝斯音軌、打擊鼓點音軌與隨機的主旋律），音樂會隨著戰鬥同步播放。

### 3. 機庫整備與持久化升級系統 (Hangar Upgrades)
玩家在關卡中擊落敵機收集的「星塵晶體」會透過 `localStorage` 持久化儲存。在「機庫整備」商店中可永久升級：
* **強化裝甲 (Max Armor)**：提升生命值上限。
* **能量護盾 (Max Shield)**：提升能量護盾容量與充電恢復速度。
* **主砲出力 (Weapon System)**：提升初始射速與彈幕威力的上限（主砲 Weapon Level 可在戰鬥中吃道具最高升至 Lv 5，解鎖多向散射雷射）。
* **磁力牽引 (Magnet Range)**：擴大自動吸引星塵晶體與道具的判定範圍。
* **炸彈艙量 (Starting Bombs)**：增加開局時配備的量子炸彈數量。

### 4. 精巧的敵機 AI 與引導飛彈
* **Scout (小型偵察機 - 綠色系)**：做弦波漂移，高速發射普通子彈。
* **Fighter (中型侵略機 - 桃紅/紫色系)**：鎖定玩家橫向追蹤並停留，發射 **Guided Missiles (紅外線引導尋標飛彈)**。飛彈會自動修正軌跡划出弧線追擊玩家，並拖曳橘色尾煙粒子。
* **Bomber (大型飛翼轟炸機 - 橘黃/金灰色系)**：重裝甲，發射重型三向擴散彈，並有機率夾雜引導飛彈壓制。
* **Boss "Void Weaver" (虛空編織者)**：
  * **第一階段**：發射 8 向旋轉花瓣彈幕。
  * **第二階段**：朝玩家狙擊發射 5 向扇形擴散重彈。
  * **第三階段**：雙側副砲反向旋轉發射螺旋彈幕，並有隨機重型狙擊彈。

## ⌨️ 遊戲操作控制 (Controls)

為了避免按鍵與瀏覽器預設行為產生衝突，遊戲設定了高強度的鍵盤攔截與去焦點機制：

| 按鍵 | 操作動作 |
| :--- | :--- |
| `W`, `A`, `S`, `D` 或 `↑`, `←`, `↓`, `→` | 控制戰機進行 **八向平滑移動**（限制於畫布內） |
| `Space` (空白鍵) | **發射雷射**（發射前掠彈幕，點擊選單按鈕後會自動去焦點以防重複觸發） |
| `B` | **釋放量子炸彈**（全螢幕亮光、強烈畫面震動、清空所有敵彈並重創畫面上所有敵人） |
| `Escape` 或 `P` | **暫停任務** / 繼續任務 |

## 🚀 執行與部署方式 (Getting Started)

### 方式 1：直接雙擊執行 (最簡單)
由於專案不包含任何外部資源載入，您只需下載此專案，並在檔案總管中**直接雙擊 `index.html`**，即可直接在瀏覽器中啟動遊玩！

### 方式 2：使用本地 HTTP 伺服器
如果您希望透過網頁伺服器開啟，可在專案根目錄下執行：
```bash
# 安裝並啟動簡易伺服器
npx http-server ./
```
啟動後在瀏覽器造訪 `http://localhost:8080` 即可開始遊戲。
